# Proxytools
A program written based on socket, can be used with sqlmap and other tools,
usge:
python proxy.py http.txt,
http.txt is you file include ip and port,every line must like this,
1.1.1.1:8088,
2.2.2.2:8089,
3.3.3.3:9999,
